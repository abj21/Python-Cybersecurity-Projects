Basic Port Scanner
Overview
The Basic Port Scanner is a Python-based cybersecurity tool that scans a target host to identify open ports within a specified range. Open ports indicate services that are running and accepting connections on the target system.
Features
Scan a target IP address or hostname
Detect open ports
Display scan results in the terminal
Simple and beginner-friendly implementation
Uses Python socket programming
