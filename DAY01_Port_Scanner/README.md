# Day 01 - Port Scanner

## Description

A beginner-friendly TCP Port Scanner built using Python's socket module. This project scans ports from 1 to 50 on a target host and identifies which ports are open.

As this is my first cybersecurity project in the series, I intentionally limited the scan range to 50 ports to keep the implementation simple and understand the fundamentals of socket programming and network communication. Future versions will include larger scan ranges, multi-threading, service detection, and performance improvements.

## Features

* Scans ports 1 to 50
* Uses TCP connections
* Detects open ports
* Built using Python's built-in socket module
* Beginner-friendly implementation

## Technologies Used

* Python 3
* Socket Programming

## Concepts Learned

* Socket Programming
* IPv4 Addressing
* TCP Communication
* Port Scanning Fundamentals
* Loops and Conditional Statements

## Current Limitations

* Scans only ports 1-50
* Single-threaded implementation
* No service detection
* Slower than professional tools like Nmap

## Future Improvements

* Custom port ranges
* Multi-threading
* Service detection
* Faster scanning
* Better error handling
* Export scan results

This project was created for educational purposes and should only be used on systems you own or have permission to test.
