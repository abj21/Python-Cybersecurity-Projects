# Day 08 - Log Analyzer

## Overview

This project is a simple Log Analyzer built using Python.

It reads a log file, extracts IP addresses, counts requests, and displays the most active IP addresses.

This project helped me learn:
- File Handling
- Lists
- Loops
- Python Collections (Counter)
- Basic Log Analysis

---

## Features

- Read log files
- Extract IP addresses
- Count total requests
- Count unique IP addresses
- Display request count for each IP
