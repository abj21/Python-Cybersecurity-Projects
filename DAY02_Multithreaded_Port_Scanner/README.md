# Day 02 - Multithreaded Port Scanner

## Description

A Python-based multithreaded port scanner that scans a target host for open ports within a user-defined port range.

## Features

* Multithreaded scanning for improved speed
* Supports both IP addresses and domain names
* User-defined start and end ports
* Detects open ports
* Displays scan results and execution time
* Beginner-friendly cybersecurity project

## Example Usage

Enter IP Address or Domain: scanme.nmap.org

Enter Start Port: 20

Enter End Port: 100

The scanner will check all ports between the specified start and end values and display any open ports found.

## Technologies Used

* Python
* socket
* threading
* queue

## Educational Purpose

This project was created for learning network reconnaissance and multithreaded programming concepts in Python.
